Ready to Download Dapper Linux?

Live images for Dapper Linux are found on this page. At the moment, Dapper Linux only supports computers with 64 bit processors. Don't worry, you will most likely have one.

Need help? Read the Installation instructions, How to make a bootable usb or the Release notes.

Downloading Dapper Linux because of a strong need for security and privacy? You may want to read the Threat model first to see what threats Dapper Linux is designed for, and to see what is not covered.

The below images are live images. They let you try Dapper Linux without erasing anything, simply load onto a USB stick and test it out. If you decide Dapper Linux is right for you, you can install directly to your computer from the live desktop.
